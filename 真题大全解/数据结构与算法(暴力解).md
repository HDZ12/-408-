暴力解法：构造一个新的数组，直接赋值。
空间复杂度: $O(n)$ ,时间复杂度：$O(n)$
```C
void remove_p(int a[], int p, int n) {
    int array[n];
    int i = p;
    int j = 0;
    for (i = p; i < n; i++) {
        array[j++] = a[i];
    }
    for (int k = 0; k < p; k++) {
        array[j++] = a[k];
    }
}
```
